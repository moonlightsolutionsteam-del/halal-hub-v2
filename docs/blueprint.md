# **App Name**: HalalSphere

## Core Features:

- Halal Restaurant Listings: Display a list of halal restaurants with location and detailed information. Businesses can claim and manage their listing. Halal status is determined via user submitted documentation and review, so that listings represent crowd-sourced guidance only, with no guarantee.
- Halal Product Verification Tool: Users can submit product photos and details. An AI tool checks ingredient lists and certifications, returning a confidence score about halal compliance, indicating what aspects of the item caused concern or compliance. If compliance cannot be definitively proven, it must be communicated to the end user. Users are warned of the risks associated with this unvalidated information.
- Prayer Time Notifications: Provide prayer time notifications based on user location.
- Qibla Direction: Offer a Qibla direction finder using the device's compass and location.
- Halal Travel Guide: Create a guide that suggests halal-friendly places, restaurants, and mosques nearby.
- Community Forum: Enable users to discuss halal products, restaurants, and issues within the community.
- Halal Events: List halal-friendly events such as seminars, lectures or celebrations

## Style Guidelines:

- Primary color: Emerald green (#50C878) to evoke feelings of trust and nature.
- Background color: Soft, desaturated green (#E8F5E9).
- Accent color: Golden yellow (#FFC107) to highlight important information and CTAs.
- Body and headline font: 'PT Sans' sans-serif, for legibility and a modern feel.
- Use minimalist icons that are culturally sensitive and universally understandable.
- A clean and intuitive layout that focuses on user accessibility, with well-spaced elements.
- Subtle animations, like gentle transitions and feedback, for a smooth user experience.
